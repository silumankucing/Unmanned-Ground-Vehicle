Open source unmanned ground vehicle with arm manipulator built for learning purposes about robotics, computer vision and autonomous and R/C systems<br>

<ins>*Hardware*</ins>
- Raspberry Pi 4 <br>
- Arduino Uno <br>
- Arduino Nano <br>
- Themperature Sensor
- 5MP Camera Logitech
- GPS Module
- Vibration Module
- IMU62000
- 5V DC Motor and 3 Inch Wheel
- Servo D8120
- Servo 9SG
- LiPo Battery
- Step Down 5V
- Motor Driver L298N
- Limit Switch
- Ultrasonic Module

<ins>*Software*</ins>
- Ubuntu ARM
- ROS
- Python
- Arduino

<ins>*Mechanical*</ins>
- 3D printed PLA <br>
